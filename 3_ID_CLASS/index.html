ID e CLASS

ID

o ID funciona como um número de CPF, cade elemento só pode ter um valor de ID. O CPF é único, ou seja, niguem pode ter um CPF igual ao meu. Isso significa que o ID também é único, então não é possivel repetir o mesmo valor duas vezes.

CLASS

O CLASS funciona de maneira oposta do ID. Diferente do ID que só pode ter um valos, np CLASS você pode ultilizar o mesmo valor para vários elementos.
